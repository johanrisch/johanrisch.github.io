ezRest

ezRest is an Android library that has two components, one module that let's you post tasks on background threads, derived from ICDispatch, combined with a fully fledged framework for sending http requests.

The paginator object can be removed from the arguments, the only required argument is either "controlDiv", the id of the container to hold all of the toggle, set and filter elements. In the simple demo the table has not been marked with any datafields, they are generated by ezList. It is possible to markup the table with data to inform ezList on how to behave. This can be seen in the advanced configuration demo.
As you can see there are three types of controls. Toggle buttons (generated by ezList), set buttons and filter fields (defined by you). There really is no limit to what you can do in these filters. Be sure to take a look at the API documentation on gihub to see the full potential.
